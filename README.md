# 台中安全路線導航 SafeTaichung

**課程名稱**：計算思維與人工智慧
**研究題目**：台中市都市犯罪分析：以 AI 輔助打造安全永續城市

---

## 專案簡介

本專案結合臺中市竊盜逐案開放資料與人口統計，透過行政區風險指標與時段分析，設計出「台中安全路線導航」原型系統，協助市民與遊客在規劃移動路線時兼顧距離與治安風險。

### 核心功能

1. **治安風險地圖**：以行政區為單位顯示竊盜風險，可切換「總件數」與「每萬人竊盜率」
2. **安全路線規劃**：輸入起點、終點與時間，評估路線風險等級
3. **安全建議卡片**：依據路線與時段提供個人化安全提示

## 分析方法

### 風險指標計算

1. **每萬人竊盜率** = 總竊盜件數 / 人口 × 10,000
   - 避免人口大區件數自然較多的生態謬誤

2. **時段風險分數** = 該時段案件數 / 全日平均案件數
   - 分數 > 1 表示該時段風險高於平均

3. **路線風險分數** = Σ(行政區竊盜率 × 時段風險分數) / 經過區數
   - 綜合考量空間與時間風險

### 風險等級分類

| 等級 | 每萬人竊盜率 | 顏色 |
|------|-------------|------|
| 低   | ≤ 33% 分位數 | 🟢 綠色 |
| 中   | 33-67% 分位數 | 🟡 橘色 |
| 高   | > 67% 分位數 | 🔴 紅色 |

## Demo 情境

### 情境 A：大學生晚上從台中車站前往逢甲夜市

- **輸入**：起點「台中車站」、終點「逢甲夜市」、時間 22:00
- **預期結果**：
  - 經過行政區：中區 → 西屯區
  - 中區夜間風險偏高（每萬人竊盜率 56.7）
  - 建議結伴同行、注意機車停放

### 情境 B：白天從勤美誠品綠園道前往一中街商圈

- **輸入**：起點「勤美誠品綠園道」、終點「一中街商圈」、時間 15:00
- **預期結果**：
  - 經過行政區：西區 → 北區
  - 白天時段風險相對較低
  - 風險等級：中

## 使用限制

1. **本系統為學術/教學用途**，風險分數為相對指標，實際治安仍以官方資訊為準
2. 開放資料僅涵蓋四類竊盜（機車、汽車、住宅、自行車），非完整竊盜統計
3. 資料時間範圍為 105-108 年（2016-2019），較近期資料尚未開放
4. 路線規劃為簡化版本，未實際串接 Google Maps 導航 API
